Decision Tree pour la prédiction de type des fleurs - IRIS

Détails

IRIS est un jeu de données bien connu en ML, qui contient des mesures de différentes caractéristiques de fleurs d'iris.

Le modèle Decision Tree atteint une précision de 100% sur ce dataset, ce qui démontre sa capacité à créer des règles de décision parfaites pour ce problème.

Les étapes d'implémentation:

  • Chargement du jeu de données IRIS
  • Prétraitement des données
  • Entraînement du modèle Decision Tree
  • Évaluation du modèle
  • Enregistrement du modèle entraîné

Performance du Modèle

Précision Globale

100%

Importance des Caractéristiques
  • Petal Length: 93.2% (Très importante)
  • Petal Width: 6.8% (Importante)
  • Sepal Length: 0% (Non utilisée)
  • Sepal Width: 0% (Non utilisée)
Info: Le modèle entraîné sauvegardé dans cet atelier est celui utilisé dans le test (Tester le modèle). L'arbre de décision a automatiquement identifié que seules les dimensions des pétales sont nécessaires pour classifier parfaitement les espèces d'iris.
Avantage du Decision Tree: Contrairement au Random Forest, vous pouvez facilement interpréter les règles de décision utilisées par le modèle !
Tester le Modèle Decision Tree

Règles de Décision Simplifiées

Si Petal Length ≤ 2.45 cm → Setosa
Sinon si Petal Width ≤ 1.65 cm → Versicolor
Sinon → Virginica
