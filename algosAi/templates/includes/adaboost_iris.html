AdaBoost pour la prédiction de type des fleurs - IRIS

Détails

IRIS est un jeu de données bien connu en ML, qui contient des mesures de différentes caractéristiques de fleurs d'iris.

Le modèle AdaBoost atteint une précision de 93.3% sur ce dataset, en utilisant des arbres de décision comme weak learners.

Les étapes d'implémentation:

  • Chargement du jeu de données IRIS
  • Prétraitement des données
  • Entraînement du modèle AdaBoost avec 100 estimateurs
  • Évaluation du modèle
  • Enregistrement du modèle entraîné

Performance du Modèle

Précision Globale

93.3%

Importance des Caractéristiques
  • Petal Width: 64.9% (Très importante)
  • Petal Length: 26.0% (Importante)
  • Sepal Length: 5.5% (Faible impact)
  • Sepal Width: 3.6% (Très faible impact)
Info: Le modèle entraîné sauvegardé dans cet atelier est celui utilisé dans le test (Tester le modèle). AdaBoost combine 100 arbres de décision faibles pour créer un classificateur robuste qui se concentre progressivement sur les échantillons difficiles.
Avantage d'AdaBoost: L'algorithme s'adapte automatiquement aux échantillons mal classifiés, améliorant progressivement les performances !
Tester le Modèle AdaBoost

Processus d'Apprentissage AdaBoost

Étapes séquentielles :
  1. Itération 1: Tous les échantillons ont le même poids
  2. Entraînement: Un arbre faible apprend sur les données pondérées
  3. Évaluation: Calcul de l'erreur et du poids du modèle
  4. Mise à jour: Augmentation du poids des échantillons mal classifiés
  5. Répétition: 100 fois pour créer un ensemble robuste
  6. Prédiction finale: Vote pondéré de tous les arbres

Configuration du Modèle

Paramètres AdaBoost
  • n_estimators: 100
  • learning_rate: 1.0
  • algorithm: SAMME
  • base_estimator: DecisionTree (max_depth=1)
Métriques de Performance
  • Accuracy: 93.3%
  • Validation croisée: 95.3% (±6.8%)
  • Précision par classe: Setosa (100%), Versicolor (90%), Virginica (90%)
